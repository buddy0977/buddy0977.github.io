Experiment 3
Data Pre-processing using Python on given dataset 
import pandas as pd
from sklearn.preprocessing import LabelEncoder
from sklearn.preprocessing import MinMaxScaler

# Load Dataset
data = pd.read_csv("student_practice_dataset.csv")

# Display Original Dataset
print("Original Dataset:\n")
print(data)

# Handling Missing Values
data.fillna(data.mean(numeric_only=True), inplace=True)

# Encoding Categorical Data
le = LabelEncoder()
for col in data.select_dtypes(include='object').columns:
    data[col] = le.fit_transform(data[col])

# Normalization
scaler = MinMaxScaler()
data[data.columns] = scaler.fit_transform(data[data.columns])

# Display Processed Dataset
print("\nPreprocessed Dataset:\n")
print(data)
Theory▼
Data pre-processing is an important step in the Knowledge Discovery in Databases (KDD) process. Real-world data is often incomplete, noisy, inconsistent, and contains missing values. The main purpose of pre-processing is to improve the quality of data before applying data mining algorithms. Data cleaning involves handling missing values, removing noise, correcting inconsistencies, and detecting outliers. Data integration combines data from multiple sources into a single dataset. Data transformation converts data into a suitable format through normalization, aggregation, and encoding techniques. Data reduction reduces the volume of data while maintaining important information through feature selection and dimensionality reduction. Data discretization converts continuous values into intervals or categories. Normalization techniques such as Min-Max and Z-score scaling help bring attributes to a common range. Proper pre-processing improves the efficiency and accuracy of classification, clustering, and association rule mining algorithms. It also reduces computational complexity and improves model performance. Therefore, data pre-processing is considered the foundation of successful data mining.
Experiment 4
Implementation of Classification Algorithm (Decision Tree / Naive Bayes) using Python 
import pandas as pd
from sklearn.preprocessing import LabelEncoder
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier
from sklearn.metrics import accuracy_score

# Load Dataset
data = pd.read_csv("student_practice_dataset.csv")
print("Original Dataset:\n")
print(data)

# Convert Text Data into Numbers
le = LabelEncoder()
for col in data.select_dtypes(include='object').columns:
    data[col] = le.fit_transform(data[col])

# Input and Output
X = data.drop('Result', axis=1)
y = data['Result']

# Split Dataset
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=1
)

# Create Decision Tree Model
model = DecisionTreeClassifier()

# Train Model
model.fit(X_train, y_train)

# Prediction
y_pred = model.predict(X_test)

# Accuracy
print("\nPredicted Output:")
print(y_pred)
print("\nAccuracy:")
print(accuracy_score(y_test, y_pred))
Theory▼
Classification is a supervised learning technique used to predict categorical outcomes from labeled data. A Decision Tree is a non-parametric supervised learning algorithm used for classification and regression tasks. It consists of a root node, internal nodes, branches, and leaf nodes arranged in a hierarchical structure. The tree is constructed by recursively splitting the dataset based on attribute values. This process is known as recursive partitioning. The objective is to create homogeneous subsets that belong to the same class. Decision trees are easy to understand, interpret, and visualize. They can handle high-dimensional datasets and require little domain knowledge. Naive Bayes is another classification algorithm based on Bayes' Theorem and probability theory. It assumes that all attributes are independent of each other. Prior probabilities and conditional probabilities are used to determine the final class label. Classification algorithms are widely used in medical diagnosis, spam filtering, fraud detection, and customer behavior prediction. They provide accurate predictions and assist in decision-making processes.

Implementation of Association Rule Mining Algorithm (Apriori) using Python

import pandas as pd
from apyori import apriori

# Sample Transactions
data = [
    ['Milk', 'Bread'],
    ['Bread', 'Butter'],
    ['Milk', 'Bread', 'Butter'],
    ['Bread'],
    ['Milk', 'Butter']
]

# Apply Apriori
rules = apriori(
    data,
    min_support=0.2,
    min_confidence=0.5,
    min_lift=1
)

# Display Rules
results = list(rules)
for item in results:
    print(item)
Theory▼
Association Rule Mining is a data mining technique used to discover interesting relationships among items in large datasets. It identifies patterns showing how items are associated with one another in transactions. The Apriori algorithm is one of the most popular algorithms used for association rule mining. It is based on the Apriori property, which states that every subset of a frequent itemset must also be frequent. The algorithm works iteratively to generate frequent itemsets and association rules. Market Basket Analysis is a common application of Apriori where customer purchasing behavior is analyzed. Important measures used in association rule mining are Support, Confidence, Lift, and Conviction. Support measures how frequently an itemset appears in the dataset. Confidence indicates the probability of occurrence of one item when another item is present. Lift determines the strength of the relationship between itemsets. Conviction measures the dependency of one item on another. Association rule mining helps businesses improve product placement, marketing strategies, and recommendation systems. It is widely used in retail, healthcare, and web usage mining.

Implementation of Clustering Algorithm (K-Means / Agglomerative) using Python 
import pandas as pd
from sklearn.preprocessing import LabelEncoder
from sklearn.cluster import KMeans

# Load Dataset
data = pd.read_csv("student_practice_dataset.csv")

# Convert Text Columns to Numbers
le = LabelEncoder()
for col in data.select_dtypes(include='object').columns:
    data[col] = le.fit_transform(data[col])

# K-Means Model
kmeans = KMeans(n_clusters=2, random_state=0)

# Create Clusters
data['Cluster'] = kmeans.fit_predict(data)

# Display Output
print(data)
Theory▼
Clustering is an unsupervised machine learning technique used to group similar data objects into clusters. Objects within the same cluster are more similar to each other than to objects in other clusters. K-Means is one of the simplest and most widely used clustering algorithms. The algorithm divides a dataset into K predefined clusters. It begins by selecting K initial centroids randomly. Each data point is assigned to the nearest centroid using a distance measure such as Euclidean distance. After assignment, new centroids are calculated as the mean of all points in each cluster. The assignment and centroid update steps are repeated until convergence is achieved. The objective of K-Means is to minimize the Sum of Squared Errors (SSE) within clusters. K-Means is efficient, scalable, and suitable for large datasets. It is commonly used in customer segmentation, image compression, pattern recognition, and market analysis. However, it requires the number of clusters to be specified beforehand and performs best on spherical clusters.
Experiment 7
Implementation of Prediction Algorithm (Linear Regression) using Python 
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error, mean_absolute_error

data = pd.read_csv("marks.csv")

X = data[['Hours']]
y = data['Marks']

X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.25, random_state=1
)

model = LinearRegression()
model.fit(X_train, y_train)

y_pred = model.predict(X_test)

print("Actual Values:")
print(y_test.values)
print("\nPredicted Values:")
print(y_pred)
print("\nMSE =", mean_squared_error(y_test, y_pred))
print("MAE =", mean_absolute_error(y_test, y_pred))
Theory▼
Linear Regression is a supervised learning algorithm used for predicting continuous numerical values. It establishes a linear relationship between a dependent variable and one or more independent variables. The mathematical model of simple linear regression is represented as Y = mX + c, where m is the slope and c is the intercept. The slope indicates the rate of change in the dependent variable with respect to the independent variable. Linear regression uses the Least Squares Method to determine the best-fit line. This method minimizes the sum of squared differences between actual and predicted values. The algorithm is easy to implement and interpret. It is widely used in forecasting, trend analysis, and predictive modeling. Model performance is evaluated using error metrics such as Mean Absolute Error (MAE), Mean Squared Error (MSE), and Root Mean Squared Error (RMSE). Linear regression helps identify relationships among variables and supports informed decision-making. Applications include sales forecasting, stock market prediction, and economic analysis.

Theory▼
Association rule learning is an unsupervised learning technique used to discover relationships among variables in large datasets. It identifies patterns indicating which items frequently occur together in transactions. Market Basket Analysis is a common application where retailers analyze customer purchasing behavior. Apriori is a widely used algorithm for generating frequent itemsets and association rules. The algorithm uses a level-wise search strategy and the Apriori property to reduce the search space. WEKA provides an easy-to-use environment for generating and evaluating association rules. This experiment demonstrates the practical application of Apriori for extracting hidden patterns from data.
